# Known Limitations

This document is intentionally conservative for private testing.

## Current limitations

- Source code is not publicly distributed.
- Installable ZIP builds are not published in the public GitHub repository.
- The internal QA scene is private and should not be used as public demo material.
- External testing is selective and may require direct build delivery.
- Runtime QA currently validates a controlled internal scene. Broader production-scene validation should be added as testing expands.

## Non-goals for the public repository

- This repository is not an open-source release.
- This repository is not a commercial marketplace page.
- This repository does not include the Blender add-on source.
- This repository does not include the internal runtime QA harness.
