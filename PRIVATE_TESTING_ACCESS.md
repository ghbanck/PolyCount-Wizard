# Private Testing Access

PolyCount Wizard v2 is available only as a controlled private testing preview.

## Tester expectations

Private testers should:

- use the tool only for evaluation and feedback;
- avoid redistributing the package;
- avoid uploading the package to public repositories or marketplaces;
- report Blender version, OS, reproduction steps, and screenshots when reporting bugs;
- avoid sharing internal implementation details.

## Feedback format

Use the GitHub issue template in this repository or provide:

- Blender version
- Operating system
- Scene size or rough mesh complexity
- Steps to reproduce
- Expected result
- Actual result
- Screenshot or short capture when useful
- Whether modifiers were enabled
- Whether filters were enabled
- Whether the issue affects analysis, overlays, CSV, settings, or UI
